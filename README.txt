# Movie Search Application

This React-based web application allows users to search for movies and view their details using the OMDB API. 

To set up and run the application:

1. Clone the repository: `git clone https://github.com/yourusername/movie-search-application.git`
2. Navigate to the project directory: `cd movie-search-application`
3. Install dependencies: `npm install`
4. Run the application: `npm start`

The app will be available at http://localhost:3000. It features movie search by title, detailed movie information 
display, and responsive design. It is built using React and React Router, it demonstrates the use of component-based 
architecture, state management, and API integration.
